// Privducai

// A macOS app that provides web search assistance and a context-aware chat experience using on-device models.

// Build Setup (Signing-neutral)

// This repository avoids committing any developer-specific signing settings so anyone can clone and build.

// - Simulator builds require no signing.
// - For device builds or archiving, set your own Team locally.

// Steps
// 1. Open the project in Xcode.
// 2. If building for device/archive, set your personal Team under the target's Signing & Capabilities.
// 3. Optionally create a local signing override:
//    - Copy `Configs/LocalSigning.xcconfig.sample` to `Configs/LocalSigning.xcconfig`.
//    - Fill in your `DEVELOPMENT_TEAM` and any other signing keys you prefer.

// Notes
// - User-specific Xcode data and the local signing override file are ignored by git.
// - If signing keys accidentally get committed, run:
//   ```bash
//   ./scripts/clean_signing.sh
//   ```
//   Then review and commit the changes.
